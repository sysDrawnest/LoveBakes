# LoveBakes Website

Official website and ordering platform for **LoveBakes** – a homemade bakery brand focused on fresh cakes, pastries, cookies, and custom dessert creations.

This project powers the digital storefront of the brand, allowing customers to explore the menu, place orders, request custom cake designs, and connect with the bakery for delivery or pickup.

## Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Authentication:** JWT
* **Other Integrations:** Email notifications, WhatsApp order alerts, payment gateway

## Features

* Online bakery menu and product pages
* Custom cake request system
* Shopping cart and checkout
* User accounts and order history
* Admin dashboard for product and order management
* Email and WhatsApp order notifications
